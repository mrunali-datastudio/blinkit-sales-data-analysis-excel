# Blinkit Sales Data Analysis Dashboard (Excel)

📊 Project Overview
This project focuses on analyzing Blinkit sales data using Microsoft Excel. It includes a KPI dashboard and detailed pivot table analysis to understand sales performance, customer preferences, and outlet-level insights.

🔑 Key Features
- KPI Metrics: Total Sales, Average Sales, Number of Items, Average Rating
- Sales by Item Type (Fruits, Snacks, Dairy, etc.)
- Fat Content Analysis (Low Fat vs Regular)
- Sales by Outlet Size (Small, Medium, High)
- Sales by Outlet Type (Supermarket Type1, Type2, Type3, Grocery Store)
- Outlet Establishment Trends

📈 Pivot Table Analysis
- Item-wise sales breakdown using Pivot Tables
- Outlet tier analysis (Tier 1, Tier 2, Tier 3)
- Fat content comparison across outlets
- Matrix analysis for outlet types with total sales
- Summary tables for better decision-making

🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Visualization

📌 Key Insights
- Fruits and Snacks categories generate high sales
- Tier 3 outlets contribute significantly to revenue
- Regular fat products outperform low-fat items
- Medium-sized outlets show stable performance
